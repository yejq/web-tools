# 一些网络工具 #

语言：java

框架：wegmagic、selenium；

资源：

1. 安装最新版的chrome浏览器；
2. 下载[chromedriver.exe](http://chromedriver.storage.googleapis.com/index.html)；
3. 修改/web-tools/src/main/resources/placeholder.properties中的webdriver.chrome.driver为你本地路径；

# 实现功能 #
